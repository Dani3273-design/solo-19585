# 人机对战五子棋游戏

一个基于Python和Pygame开发的五子棋游戏，支持人机对战，包含多种AI难度等级。

## 功能特性

- **完整的游戏流程**：主页 → 设置 → 选棋 → 游戏 → 结算
- **三种AI难度**：
  - 等级1：仅考虑横竖方向拦截，不考虑斜线
  - 等级2：每3轮出现一次失误（只进攻不防守）
  - 等级3：完美防守，玩家绝无胜利可能，要么AI胜要么平局
- **15秒倒计时**：每步限时15秒，超时判负
- **落子音效**：支持开关控制
- **多线程设计**：UI独立线程，胜负检查、AI思考在后台线程运行，界面流畅不卡顿
- **美观的界面**：采用木纹棋盘风格，按钮有悬停效果

## 模块说明

### main.py
程序入口，创建UI线程并启动游戏。

### game/ui.py
- `GomokuUI` 类：游戏主界面控制器
- `Button` 类：通用按钮组件
- 实现了5个页面：主页、设置页、选棋页、游戏页、结算页
- 处理用户输入、落子逻辑、页面切换
- 集成字体加载（支持中文显示）、音效播放

### game/check.py
- `GameChecker` 类：游戏状态检查器
- 检查横、竖、斜四个方向是否有五子连珠
- 检查平局（棋盘满或无获胜可能）
- 支持异步多线程检查，不阻塞UI

### game/count.py
- `GameCounter` 类：游戏计数器
- 15秒倒计时功能，超时回调
- 回合数统计
- 总耗时统计
- 线程安全设计

### game/setting.py
- `Settings` 类：设置管理器
- AI难度等级设置（1-3级）
- 落子声音开关
- 配置持久化到JSON文件

### game/ai.py
- `GomokuAI` 类：AI下棋逻辑
- 基于评分的位置评估算法
- 三种难度等级实现
- 支持异步AI思考，不阻塞UI

## 技术要求

- Python 3.9+
- Pygame 2.5.0+
- NumPy 1.24.0+

## 安装运行

```bash
cd main
pip install -r requirements.txt
python main.py
```

## 游戏规则

1. 黑棋先行，双方交替落子
2. 先将五子连成一线（横、竖、斜）者获胜
3. 每步限时15秒，超时判负
4. 棋盘下满或双方都无获胜可能则平局

## 操作说明

1. 点击"开始游戏"进入选棋界面
2. 选择执黑棋或白棋（黑棋先行）
3. 点击棋盘交叉点落子
4. 点击"返回主页"可随时退出游戏
5. 游戏结束后可选择"再来一局"或进入"设置"

## 多线程设计

游戏采用多线程架构，确保界面流畅：

- **UI主线程**：处理Pygame事件循环、绘制界面
- **倒计时线程**：每步15秒计时，超时触发回调
- **胜负检查线程**：每次落子后异步检查胜负
- **AI思考线程**：AI计算落子位置在后台运行

线程间通过Pygame自定义事件（USEREVENT）通信，确保线程安全。
